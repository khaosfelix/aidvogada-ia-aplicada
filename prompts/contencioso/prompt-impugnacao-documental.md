# Prompt — Impugnação documental

Analise individualmente cada documento relevante.

Para cada documento, entregue:

1. ID.
2. Nome.
3. Tipo.
4. Conteúdo.
5. O que comprova.
6. O que não comprova.
7. Ausência de metadados, assinatura, contexto ou integralidade.
8. Contradições.
9. Limites probatórios.
10. Pedido de valoração restrita, quando cabível.

Não impugne documentos de forma genérica quando houver documento relevante que exija análise individual.
