# Prompt — Análise de documentos

Analise os documentos apresentados e organize:

1. Lista de documentos.
2. Tipo de documento.
3. Data.
4. Origem.
5. Partes envolvidas.
6. Conteúdo relevante.
7. Obrigações identificadas.
8. Riscos.
9. Inconsistências.
10. O que o documento comprova.
11. O que o documento não comprova.
12. Documentos complementares necessários.
13. Providências recomendadas.
