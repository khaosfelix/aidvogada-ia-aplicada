# Prompt — Memorando jurídico

Elabore memorando jurídico com:

1. Destinatário.
2. Assunto.
3. Contexto.
4. Pontos de atenção.
5. Riscos.
6. Providências.
7. Recomendação objetiva.
8. Pendências.
9. Próximos passos.

Use linguagem clara, executiva e tecnicamente segura.
