# Prompt — Relatório jurídico

Elabore relatório com:

1. Objetivo.
2. Escopo.
3. Metodologia.
4. Documentos analisados.
5. Achados relevantes.
6. Riscos.
7. Recomendações.
8. Pendências.
9. Próximos passos.
10. Anexos sugeridos.

O relatório deve facilitar tomada de decisão.
