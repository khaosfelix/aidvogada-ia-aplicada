# Exemplo fictício — Análise documental

> Exemplo meramente demonstrativo, sem dados reais.

## Documento 1 — Contrato de prestação de serviços

| Critério | Análise |
|---|---|
| Tipo | Contrato |
| Data | Não informada |
| Partes | Empresa e prestador |
| Conteúdo | Prestação por demanda |
| Obrigações | Entregas específicas |
| Risco | Cláusulas de controle excessivo |
| Lacuna | Ausência de cláusula de substituição |
| Providência | Revisar autonomia e forma de execução |
