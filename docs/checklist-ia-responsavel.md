# Checklist de uso responsável de IA no jurídico

## Sigilo e dados

- [ ] Remover dados pessoais desnecessários.
- [ ] Evitar exposição de informações sensíveis.
- [ ] Avaliar necessidade de anonimização.
- [ ] Respeitar sigilo profissional.
- [ ] Verificar se há dados de terceiros.

## Qualidade jurídica

- [ ] Conferir fatos.
- [ ] Conferir documentos.
- [ ] Conferir fundamentos.
- [ ] Conferir legislação atualizada.
- [ ] Conferir jurisprudência.
- [ ] Conferir aderência ao caso concreto.

## Alucinação

- [ ] Não aceitar artigos legais sem conferência.
- [ ] Não aceitar jurisprudências sem conferência.
- [ ] Não aceitar fatos não documentados como certeza.
- [ ] Sinalizar lacunas.
- [ ] Usar marcadores quando houver falta de fonte.

## Revisão humana

- [ ] Revisar estratégia.
- [ ] Revisar premissas.
- [ ] Revisar linguagem.
- [ ] Revisar riscos.
- [ ] Revisar conclusão.
- [ ] Validar antes de uso externo.

## Transparência interna

- [ ] Registrar limites da análise.
- [ ] Registrar documentos utilizados.
- [ ] Registrar pendências.
- [ ] Registrar pontos que dependem de confirmação.
