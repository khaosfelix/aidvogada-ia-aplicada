# Impugnação documental

Este documento organiza critérios para análise e impugnação de documentos em peças processuais e manifestações.

## Critérios de análise

Para cada documento relevante, verificar:

- ID;
- nome do documento;
- tipo de documento;
- data;
- origem;
- conteúdo;
- finalidade;
- autenticidade;
- integralidade;
- contexto;
- assinatura;
- metadados;
- cadeia de custódia;
- contradições;
- aderência ao pedido;
- o que comprova;
- o que não comprova;
- limites de força probatória.

## Estrutura sugerida

```text
Documento: [identificação]
Conteúdo: [síntese objetiva]
O que demonstra: [fato comprovado]
O que não demonstra: [limite probatório]
Fragilidades: [ausência de assinatura, contexto, metadados, integralidade etc.]
Conclusão: [valoração pretendida]
```

## Exemplos de cautela

- Documento unilateral não comprova, sozinho, fato controvertido.
- Print sem metadados deve ser analisado com cautela.
- Atestado médico não comprova automaticamente nexo causal.
- CTPS comprova vínculos formais, mas não comprova jornada.
- Comprovante de pagamento isolado deve ser contextualizado.
