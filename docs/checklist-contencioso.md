# Checklist contencioso

## Identificação inicial

- [ ] Identificar tipo de peça.
- [ ] Identificar rito.
- [ ] Identificar partes.
- [ ] Identificar competência.
- [ ] Verificar prazos.
- [ ] Verificar documentos essenciais.

## Pedidos e causa de pedir

- [ ] Listar pedidos expressos.
- [ ] Separar pedidos principais e acessórios.
- [ ] Identificar pedidos genéricos.
- [ ] Identificar pedidos sem causa de pedir.
- [ ] Identificar alegações sem pedido autônomo.
- [ ] Marcar temas que não devem virar tópico autônomo.

## Provas

- [ ] Mapear documentos da parte autora.
- [ ] Mapear documentos defensivos.
- [ ] Mapear documentos ausentes.
- [ ] Identificar fatos incontroversos.
- [ ] Identificar fatos controvertidos.
- [ ] Identificar contradições.
- [ ] Identificar lacunas probatórias.

## Estratégia

- [ ] Definir preliminares.
- [ ] Definir prejudiciais.
- [ ] Definir mérito por pedido existente.
- [ ] Definir impugnações documentais.
- [ ] Definir provas a produzir.
- [ ] Definir riscos.
- [ ] Definir pedidos finais.

## Revisão

- [ ] Conferir fundamentos legais.
- [ ] Conferir jurisprudências.
- [ ] Conferir linguagem.
- [ ] Conferir coerência entre fato, prova e pedido.
- [ ] Conferir se não há fato afirmado sem prova.
- [ ] Conferir se não há pedido inexistente combatido autonomamente.
