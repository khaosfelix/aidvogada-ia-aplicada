# Checklist consultivo

## Entrada

- [ ] Identificar a pergunta jurídica.
- [ ] Identificar o objetivo da consulta.
- [ ] Identificar destinatário.
- [ ] Identificar urgência.
- [ ] Identificar documentos disponíveis.
- [ ] Identificar documentos pendentes.

## Premissas

- [ ] Separar fatos documentados.
- [ ] Separar fatos informados, mas não comprovados.
- [ ] Registrar limites da análise.
- [ ] Registrar hipóteses assumidas.
- [ ] Registrar pontos que exigem validação.

## Análise

- [ ] Identificar normas aplicáveis.
- [ ] Identificar obrigações.
- [ ] Identificar riscos.
- [ ] Identificar alternativas.
- [ ] Identificar providências.
- [ ] Identificar impacto operacional.

## Entrega

- [ ] Responder objetivamente à consulta.
- [ ] Indicar riscos.
- [ ] Indicar providências.
- [ ] Indicar documentos necessários.
- [ ] Indicar pontos de cautela.
- [ ] Evitar conclusão sem base documental suficiente.

## Revisão final

- [ ] Conferir coerência.
- [ ] Conferir linguagem.
- [ ] Conferir fundamentos.
- [ ] Conferir limites.
- [ ] Conferir se a resposta é prática e utilizável.
