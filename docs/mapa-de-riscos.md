# Mapa de riscos

## Objetivo

Organizar riscos jurídicos, processuais, documentais, probatórios, consultivos e operacionais.

## Categorias

| Categoria | Exemplos |
|---|---|
| Processual | prazo, competência, rito, inépcia, ônus da prova |
| Probatório | ausência de documento, prova frágil, contradição |
| Jurídico | tese controvertida, norma aplicável, entendimento divergente |
| Consultivo | obrigação indefinida, lacuna de premissa, documentação insuficiente |
| Operacional | dificuldade de implementação, custo, impacto interno |
| Reputacional | exposição pública, conflito sensível, impacto institucional |
| Financeiro | condenação, multa, contingência, custo de adequação |

## Classificação

- Baixo: risco controlado, com documentação suficiente.
- Médio: risco relevante, mas mitigável.
- Alto: risco provável ou com impacto significativo.
- Crítico: risco imediato, com alta exposição ou urgência.
