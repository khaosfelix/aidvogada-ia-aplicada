# Contribuição

Este é um projeto autoral em desenvolvimento.

Sugestões de melhoria podem envolver:

- documentação;
- fluxos jurídicos;
- prompts;
- exemplos fictícios;
- checklists;
- arquitetura de informação;
- governança de IA;
- interface futura.

## Diretrizes

- Não incluir dados reais de clientes, partes ou processos.
- Não incluir informações sigilosas.
- Não incluir jurisprudência ou legislação sem conferência.
- Não utilizar documentos sensíveis.
- Priorizar exemplos fictícios e anonimizados.
